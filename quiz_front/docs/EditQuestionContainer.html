Component for displaying question creation or edition. It is smart component which on answer button click will send request on backend

Source:

Example
const answers = [{id: 100, text: 'apple', correct: true}, {id: 200, text: 'banana', correct: false}]
const question = {id: 121, text: 'Which fruit is green?', answers}
const onSuccess = id => console.log("Question was modified")

return (
  <EditQuestionContainer
     question={question}
     onSuccess={onSuccess}
  />
)
